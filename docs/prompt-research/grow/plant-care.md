# Grow - 栽培方法プロンプトリサーチ

## 概要

自然農法を中心とした栽培方法を調べるためのプロンプト。

## プロンプト一覧

### 1. care_natural - 自然農法での育て方

**目的**: 化学肥料・農薬を使わない栽培方法を学ぶ

#### 現行プロンプト（v1.0）

```
{plant}を自然農法（無農薬・無化学肥料）で育てたいです。

以下について教えてください：

1. 土づくり（堆肥、緑肥など）
2. 種まき・定植のコツ
3. 水やり・管理のポイント
4. 自然な病害虫対策
5. コンパニオンプランツ
6. 収穫のタイミング

※できれば福岡正信、川口由一、木村秋則などの日本の自然農法の知見も含めてください。
```

#### 設計意図

- **「自然農法」を明確に定義**: 無農薬・無化学肥料と明記
- **包括的な情報**: 種まきから収穫までの全工程
- **日本の自然農法の先人を指定**: 日本独自の知見を引き出す

#### 日本の自然農法について

| 人物 | 農法名 | 特徴 |
|------|--------|------|
| 福岡正信 | 自然農法 | 不耕起、無肥料、無農薬、無除草 |
| 川口由一 | 自然農 | 耕さない、草と虫を敵としない |
| 木村秋則 | 自然栽培 | 無農薬りんご、土壌微生物重視 |
| 岡田茂吉 | MOA自然農法 | 自然尊重、土を生かす |

#### 改善検討事項

- [ ] 農法の流派を指定できるようにする？
- [ ] ユーザーの経験レベルに応じた回答を得られるか？
- [ ] 地域の気候条件も考慮すべきか？

### 2. care_trouble - 生育トラブルの解決

**目的**: 植物の異常を診断し、自然な対策を見つける

#### 現行プロンプト（v1.0）

```
{plant}を育てていますが、以下の症状が出ています：

【症状】
{symptoms}

【栽培環境】
- 場所：{location}
- 農法：{farming_method}
- 最近の天候：{weather}

考えられる原因と、自然な方法での対策を教えてください。
```

#### 設計意図

- **症状の詳細を記述**: AIが診断しやすい情報を提供
- **環境情報を含める**: 原因特定の精度を上げる
- **自然な方法を強調**: 農薬に頼らない対策

#### 改善検討事項

- [ ] 症状の表現をガイドする選択肢があると良い
- [ ] 写真アップロード対応（マルチモーダル）を案内
- [ ] 緊急度の判定も含める

## プロンプト設計のベストプラクティス

### 1. 自然農法のコンテキストを設定

```
私は化学肥料や農薬を使わない自然農法で野菜を育てています。
```

この一文を入れることで、AIは自然農法の文脈で回答する。

### 2. 日本の知見を引き出す

```
日本の自然農法（福岡正信の不耕起栽培、川口由一の自然農など）の
考え方も参考にしてください。
```

日本発の農法は海外AIでは認識が薄い場合があるため、明示的に指定。

### 3. 実行可能なアドバイスを求める

```
具体的に何をすれば良いか、手順を教えてください。
```

抽象的な説明ではなく、実際のアクションを求める。
