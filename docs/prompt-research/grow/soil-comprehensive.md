# Grow - 土壌総合プロンプトリサーチ

## 土壌の3要素

土壌を理解するには、**分類**だけでなく**3つの性質**を把握する必要がある。

### 1. 物理性（Physical Properties）

土壌の物理的な構造と特性。

| 項目 | 説明 | 観察方法 |
|------|------|----------|
| 土性（テクスチャー） | 砂・シルト・粘土の割合 | 握ってみる（さらさら/粘る） |
| 団粒構造 | 土の塊のでき方 | 崩れやすさを確認 |
| 孔隙率 | 空気・水の通り道 | 水を注いで浸透速度を見る |
| 保水性 | 水を保持する力 | 乾きやすさを観察 |
| 排水性 | 余分な水が抜ける力 | 雨後の水たまりを確認 |
| 土壌硬度 | 硬さ・締まり具合 | 棒を刺してみる |

### 2. 生物性（Biological Properties）

土壌生態系の豊かさ。自然農法で最も重視される。

| 項目 | 説明 | 観察方法 |
|------|------|----------|
| 微生物量 | 細菌・糸状菌の量 | 土の匂い（森の香り＝良好） |
| ミミズ | 土壌動物の指標 | スコップで掘って数える |
| 根圏微生物 | 菌根菌など | 植物の根を観察 |
| 有機物分解 | 落ち葉などの分解速度 | マルチの分解を観察 |
| 土壌呼吸 | 微生物活性の指標 | CO2測定（専門的） |

### 3. 化学性（Chemical Properties）

土壌の化学的組成。

| 項目 | 説明 | 測定方法 |
|------|------|----------|
| pH | 酸性・アルカリ性 | pH試験紙、測定器 |
| EC（電気伝導度） | 塩類濃度 | EC計 |
| 窒素（N） | 葉の成長に必要 | 土壌分析 |
| リン酸（P） | 根・花・実に必要 | 土壌分析 |
| カリウム（K） | 全体の健康に必要 | 土壌分析 |
| CEC | 養分保持力 | 土壌分析 |
| 有機物含量 | 腐植の量 | 色・土壌分析 |

## プロンプト設計

### 基本プロンプト：土壌の総合診断

```
{location}で{farming_method}をしています。

土壌について、以下の3つの観点から教えてください。

【1. 物理性】
- この地域で一般的な土性（砂質/壌土/粘土質）
- 排水性・保水性の傾向
- 物理性を改善するためのアドバイス

【2. 生物性】
- 土壌生物（微生物、ミミズなど）を増やす方法
- 有機物の施用方法（堆肥、緑肥、草マルチなど）
- 自然農法で土を育てるポイント

【3. 化学性】
- この地域の土壌のpH傾向
- 不足しがちな養分と自然な補い方
- 化学肥料を使わない養分管理

※{farming_method}の考え方に沿ったアドバイスをお願いします。
```

### 観察ベース：土壌診断プロンプト

```
畑の土壌を診断してほしいです。

【観察結果】
- 場所：{location}
- 土の色：{color}
- 握った感触：{texture}（さらさら/しっとり/粘る）
- 水はけ：{drainage}（良い/普通/悪い）
- 土の匂い：{smell}（森の香り/無臭/かび臭い/悪臭）
- ミミズ：{earthworms}（多い/少ない/いない）
- 雑草の種類：{weeds}

この観察から分かる、土壌の物理性・生物性・化学性の状態と、
自然農法での改善方法を教えてください。
```

### 簡易診断：5つの質問

```
以下の5つの質問に答えて、私の畑の土壌を診断してください。

1. 場所：{location}
2. 土を握ると：{texture_test}
   □ すぐ崩れる（砂質）
   □ 軽く固まるがすぐ崩れる（壌土）
   □ 粘土のように固まる（粘土質）
3. 雨の後：{drainage_test}
   □ すぐ乾く
   □ 翌日には乾く
   □ 数日水が溜まる
4. 土の匂い：{smell_test}
   □ 森のような良い香り
   □ 特に匂いなし
   □ 嫌な匂い
5. 掘ると：{life_test}
   □ ミミズがたくさんいる
   □ ミミズが少しいる
   □ ミミズがいない

【診断してほしいこと】
- 土壌タイプ（WRB分類）
- 物理性・生物性・化学性の評価
- 自然農法での改善優先順位
- 具体的な改善方法
```

## 自然農法における土壌の考え方

### 福岡正信（自然農法）
> 「土は耕さない。自然の状態が最も良い。」

- 不耕起で土壌構造を維持
- 微生物・小動物の生態系を壊さない

### 川口由一（自然農）
> 「草も虫も敵ではない。」

- 草マルチで土を覆う
- 刈った草はその場に置く

### 木村秋則（自然栽培）
> 「土が生きていれば、植物は自然に育つ。」

- 土壌微生物の活性化が鍵
- 根圏の生態系を大切に
