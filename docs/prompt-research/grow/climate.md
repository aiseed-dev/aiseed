# Grow - 気候プロンプトリサーチ

## 概要

ユーザーが自分の栽培場所の気候について調べるためのプロンプト。

土壌と比較すると個別的な項目が少なく、地域名から比較的正確な情報が得られる。

## 気候の主要項目

| 項目 | 説明 | 栽培への影響 |
|------|------|-------------|
| 気温（年間・季節） | 平均気温、最高・最低 | 作物選択、播種時期 |
| 霜日（初霜・終霜） | 霜の降りる時期 | 栽培可能期間を決定 |
| 降水量 | 年間・季節別 | 水やり計画、病気リスク |
| 梅雨 | 時期と長さ | 病害虫対策の重要期間 |
| 台風 | シーズンと頻度 | 支柱補強、収穫タイミング |
| 日照時間 | 季節別 | 作物の生育に影響 |

## プロンプト一覧

### 1. climate_comprehensive - 地域の気候を調べる

**目的**: 地域の気候を総合的に把握する

#### 現行プロンプト（v1.0）

```
{location}で家庭菜園・自然農法をしています。

この地域の気候について、以下を教えてください：

【1. 気候の基本情報】
- 気候区分（ケッペン分類や日本の気候区分）
- 年間平均気温、最高・最低気温
- 年間降水量と降雨パターン

【2. 栽培カレンダーの目安】
- 最終霜日・初霜日の目安
- 栽培可能期間（霜なし期間）
- 梅雨の時期と特徴
- 台風シーズン

【3. 季節ごとの特徴】
- 春：遅霜リスク、植え付け開始時期
- 夏：猛暑日、水やりの注意点
- 秋：収穫適期、秋まきの時期
- 冬：寒さ対策、越冬できる野菜

【4. 自然農法での気候活用】
- この気候に適した野菜・作物
- 季節の変化を活かした栽培のコツ
- 気候の課題と自然な対策
```

#### 設計意図

- **4つのセクション**: 情報を整理して把握しやすく
- **栽培カレンダーの目安**: 播種・定植の計画に直結
- **自然農法での活用**: Growのターゲットユーザーに沿った情報

---

### 2. climate_calendar - 栽培カレンダーを作る

**目的**: 特定の作物の栽培スケジュールを調べる

#### 現行プロンプト（v1.0）

```
{location}で{plant}を育てたいです。

この地域の気候に合った栽培カレンダーを教えてください：

1. 播種・定植の適期（月）
2. 収穫時期の目安
3. 生育適温
4. この地域特有の注意点（霜、梅雨、台風など）
5. おすすめの品種（この気候に強いもの）
```

#### 設計意図

- **作物特化**: 植物ごとに最適な時期を調べる
- **品種推薦**: 地域に適した品種情報

---

### 3. climate_micro - 栽培環境の微気候

**目的**: ベランダや庭など特定環境の気候を分析

#### 現行プロンプト（v1.0）

```
以下の環境で野菜を育てたいです：

【栽培環境】
- 場所：{location}
- 環境タイプ：{environment}（例：南向きベランダ、北側の庭など）
- 日当たり：{sunlight}
- 風通し：{wind}

この環境の微気候の特徴と、適した野菜を教えてください。
```

#### 設計意図

- **微気候に対応**: 同じ地域でも環境で大きく異なる
- **都市農業向け**: ベランダ菜園などに有用

## 気候区分の基礎知識

### ケッペンの気候区分（日本）

| 記号 | 名称 | 分布地域 | 特徴 |
|------|------|---------|------|
| Cfa | 温暖湿潤気候 | 本州の大部分 | 四季明瞭、梅雨あり |
| Cfb | 西岸海洋性気候 | 北海道の一部 | 冷涼、梅雨なし |
| Cw | 温暖冬季少雨気候 | 内陸部 | 冬に乾燥 |
| Dfa/Dfb | 亜寒帯湿潤気候 | 北海道 | 冬が厳しい |

### 日本の農業気候区分

| 地域 | 特徴 | 主要課題 |
|------|------|---------|
| 北海道 | 冷涼、梅雨なし | 短い栽培期間 |
| 東北 | 冷涼〜温暖 | やませ（冷たい風） |
| 関東 | 温暖、乾燥気味 | 夏の高温 |
| 中部 | 地域差大 | 山間部は寒冷 |
| 近畿 | 温暖 | 梅雨の多雨 |
| 中国・四国 | 温暖〜亜熱帯 | 瀬戸内は少雨 |
| 九州 | 温暖〜亜熱帯 | 台風、梅雨 |
| 沖縄 | 亜熱帯 | 台風、塩害 |

## 自然農法と気候

### 気候を活かす考え方

> 「自然に逆らわず、季節の流れに沿う」

- **旬を大切に**: 気候に合った時期に育てる
- **無理をしない**: ハウスや加温に頼りすぎない
- **土地の特性を知る**: 微気候を観察する

### 季節ごとの自然農法のポイント

| 季節 | ポイント |
|------|---------|
| 春 | 遅霜に注意、土が温まってから定植 |
| 夏 | 草マルチで地温抑制、朝夕の水やり |
| 秋 | 種採り、緑肥の播種 |
| 冬 | 越冬野菜、土づくり（落ち葉堆肥） |
