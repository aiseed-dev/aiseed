# AIseed 事業計画書

## ビジョン
**「AIと人が共に成長するプラットフォーム」**

---

## サービス構成

| サービス | 概要 |
|---------|------|
| **Spark** | 対話で強み発見 → 学びへの入口 |
| **Grow** | 栽培・伝統野菜辞典・料理 |
| **Nurture** | 育児・教育サポート |
| **Create** | AI版WordPress（Web制作） |
| **Learn** | プログラミング・BYOA開発スクール |

---

## BYOA（Bring Your Own Agent）モデル

```
学ぶ → 教える → 育てる → 提供する
```

利用者が自分のエージェントを育て、サービス提供者になる

## 収益モデル

### 料金設定
| プラン | 月額 | 内容 |
|-------|------|------|
| ベーシック | 3,000円 | 全サービス利用 |
| スタンダード | 5,000円 | + プログラミングスクール |
| プロ | 10,000円 | + BYOA開発サポート |

| 会員数 | 平均単価 | 月収 |
|-------|---------|------|
| 50人 | 4,000円 | 20万円 |
| 100人 | 4,000円 | 40万円 |
| 200人 | 4,000円 | 80万円 |

- Claude Max: 1.5万円/月

- **利益率: 90%以上**
